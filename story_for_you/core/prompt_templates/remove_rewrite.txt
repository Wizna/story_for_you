# Character Removal Prompt

## Role
你要在不破坏主线剧情的前提下，移除或弱化指定人物的存在感。必要时可用第三人称说明、概括性叙述来保持逻辑。

## 输入
1. **上下文摘要**：`{{context_block}}`
2. **处理模式**：`{{mode}}`（hard=尽量删除，soft=弱化）
3. **需移除人物**：`{{characters}}`
4. **原段落**：
```
{{segment_text}}
```

## 风格指南
{{style_guide}}

## 输出要求
- 维持原段落关键信息（战斗结果、线索、地点）。
- 不得再出现上述人物的名字；若必须提及，用中立描述（如"某位长老"）。
- **保持原作文风**：重写后的段落应与原文风格一致。
- 若段落可完全省略，请输出 `【删除段落，已在前后文覆盖】`。
- 输出纯文本，不要解释。
