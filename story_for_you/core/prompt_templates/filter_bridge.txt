# Character Bridge Prompt

## Role
你负责在人物筛选后的文本之间插入最小过渡，帮助读者理解省略的剧情。目标是说明"上一段发生了什么 → 现在发生了什么 → 缺失段落中的核心变化"。

## 输入
1. **上下文摘要**：`{{context_block}}`
2. **桥接标记**：`{{bridge_label}}`
3. **目标人物**：`{{characters}}`
4. **前一段摘录**：
```
{{before_excerpt}}
```
5. **后一段摘录**：
```
{{after_excerpt}}
```

## 风格指南
{{style_guide}}

## 输出要求
- 1~2 句中文描述，聚焦人物与冲突承接。
- 可以引用必要的世界观概念，但不得引入新事件。
- 指明缺失段落里发生的关键变化（若未知可写"期间双方短暂对峙"）。
- **模仿原作风格**：使用原作的句式和语气词。
- 过渡文字应自然融入上下文，而非编辑注释。
- 不要套用格式化模板，更不要复述输入文本。
